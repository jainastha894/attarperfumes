# Arjan Attars Website

Static marketing site for the Arjanmal Attarchand showcasing attars, oils, and botels only heritage. Built with HTML, Tailwind CDN, and custom CSS.

## Pages
- `index.html`: Landing page with hero, featured products, testimonials, and brand highlights.
- `shop.html`: Product gallery cards with add-to-cart CTAs (front-end only).
- `about.html`: Brand story, craft process, and values.
- `heritage.html`: Timeline of heritage and milestones.
- `contact.html`: Contact form layout and location info.
- `map.html`: Embedded location map helper page.

## Assets
- `public/styles/`: Page-specific CSS (e.g., `home.css`).
- `public/images/`: Product and background imagery plus `favicon.ico`.

## Running locally
- No build step required; open `index.html` in a browser.
- Tailwind is loaded via CDN; internet is required for those styles.

## Notes
- Forms and cart buttons are presentational only; no backend wiring.
- Update copy and imagery in `public/images`/`public/styles` as needed.
